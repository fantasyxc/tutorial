## 1. vim命令 cheat sheet
### 1. 跳转
| 命令 | 使用方法 |
| --- | --- |
| $ | 跳转到行首 |
| 0（数字） | 跳转到行尾 |
| gg | 跳转到第一行 |
| GG | 跳转到最后一行 |
| 20gg | 跳转到第20行 |

### 1.2. 编辑
| 命令 | 使用方法 |
| --- | --- |
| o（小写） | 在当前行下方新建一行 |
| O（大写） | 在当前行上方新建一行 |
| dd | 剪切1行 |
| 5dd | 剪切5行 |
| p | 粘贴 |
| :/bind | 搜索bind，按n选择下一个，N选择上一个 |

### 1.3. 设置
| 命令 | 使用方法 |
| --- | --- |
| :set nu | 显示行号 |
| :set nonu | 不显示行号 |

### 1.4. 模式
| 命令 | 使用方法 |
| --- | --- |
| v | 视图模式，按照字符选择 |
| V | 视图模式，按照行选择 |
| i | 输入模式 |
